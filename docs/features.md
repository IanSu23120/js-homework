# Features

## 介紹首頁

路由：`/`

用途：

- 說明產品概念。
- 顯示目前完成的功能。
- 提供入口前往 `/planner`。

## 功能頁

路由：`/planner`

功能：

- 查看旅程列表。
- 查看旅程統計。
- 新增旅程。
- 從地圖探索推薦景點。
- 將景點加入指定旅程與日期。

## 新增旅程

欄位：

- 旅程名稱
- 目的地城市
- 出發日期
- 回程日期
- 封面顏色

輔助資訊：

- 城市地圖
- 推薦景點
- 天氣預測

若使用者勾選推薦景點，儲存旅程時會將景點加入第一天行程。

## 城市支援

目前支援：

- 日本：東京、大阪、京都
- 韓國：首爾、釜山、濟州
- 台灣：台北、台南、高雄
- 泰國：曼谷、清邁
- 越南：河內、胡志明市、峴港
- 其他：新加坡、香港、澳門、吉隆坡、峇里島、上海

城市資料來源：`src/data/cities.js`

## 推薦景點

推薦景點資料來源：`src/data/attractions.js`

每個景點包含：

- id
- city
- aliases
- name
- category
- description
- lat
- lng

## 旅程詳情

路由：`/trip/:tripId`

功能：

- 切換不同日期。
- 新增自訂行程。
- 編輯既有行程。
- 刪除行程。
- 依行程位置顯示地圖 marker。
- 點選行程或 marker 後同步聚焦。

## 匯出

目前支援一鍵複製文字摘要。邏輯位於：

- `src/utils/exportUtils.js`

後續可擴充：

- 下載 JSON
- 匯出 Markdown
- 匯出 CSV
